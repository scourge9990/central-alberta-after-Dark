Accessibility Statement

Last Updated: June 2026

Our Commitment

Central Alberta After Dark is committed to ensuring digital accessibility for people with disabilities. We strive to improve the user experience for everyone and apply relevant accessibility standards.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. These guidelines explain how to make web content more accessible for people with disabilities and more user-friendly for everyone.

Our efforts include:
  • Semantic HTML structure for screen readers
  • Keyboard navigation support
  • Color contrast ratios meeting WCAG AA standards
  • Alt text on all meaningful images
  • Focus indicators for interactive elements

Accessibility Features

Our platform includes the following accessibility features:

  • Screen Reader Support: All interactive elements have appropriate ARIA labels and semantic HTML
  • Keyboard Navigation: Users can navigate through the site using keyboard Tab, Enter, and Escape keys
  • Color Contrast: We use a yellow (#FFEB43) on blue (#003594) color scheme that provides strong contrast ratios
  • Text Resizing: The site supports text scaling up to 200% without loss of functionality
  • Alt Text: All images include descriptive alt text
  • Form Labels: All form inputs have associated labels for screen readers
  • Focus Indicators: Interactive elements clearly show focus state for keyboard users

Known Limitations

While we strive for accessibility, you may encounter some limitations:

  • Some older third-party content (advertisements) may not be fully accessible
  • User-generated profile photos may not always have alt text descriptions
  • The live chat/community feed refreshes automatically, which may affect screen reader users

We are continuously working to improve these areas.

Feedback and Contact

We welcome your feedback on the accessibility of Central Alberta After Dark. If you encounter any barriers or have suggestions for improvement, please contact us:

  • Email: support@centralalbertaafterdark.com
  • Response Time: We aim to respond within 30 days

We will work with you to provide information in an accessible format or address any accessibility concerns.

Technical Information

Accessibility of Central Alberta After Dark is based on:

  • WCAG 2.1 Level AA standards
  • WAI-ARIA 1.1 specifications
  • HTML5 semantic markup

Assessment Approach

Our accessibility is evaluated through:

  • Self-assessment and manual testing
  • Keyboard navigation testing
  • Screen reader compatibility testing
  • Color contrast verification
